# AI Study Focus Analyzer

This project detects whether a user is focused or distracted during study using webcam input.

## Features
- Face detection using OpenCV
- Eye closure detection
- Focus vs distraction classification
- Tracks focused and distracted time

## Technologies Used
- Python 3.13
- OpenCV
- NumPy

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the project:
   python focus_detector.py

## Use Case
Helps students analyze and improve study focus using computer vision.
